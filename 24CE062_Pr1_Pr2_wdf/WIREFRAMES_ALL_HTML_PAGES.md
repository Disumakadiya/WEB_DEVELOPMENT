## COMMON LAYOUT PATTERNS IDENTIFIED:

### Header Structure (Consistent across all pages):
- University logo on the left
- "CHARUSAT UNIVERSITY" title in center
- Grid-based layout

### Navigation Patterns:
1. **Public Navigation**: Home | About | Login | Register + Enquire box
2. **Student Navigation**: Home | Profile | Events | Feedback | Logout
3. **Admin Navigation**: Home | Dashboard | Profile

### Content Areas:
- **Forms**: Login, Register, Feedback - Centered layout with input fields
- **Cards**: Dashboard, Events, Results - Grid layout with information cards  
- **Profile**: Two-column layout with photo and details
- **Landing**: Hero images and content sections

### Footer (Consistent):
- Copyright notice
- University branding
- Service portal identification

### Interactive Elements:
- Buttons for actions (Login, Register, Submit, etc.)
- Navigation links
- Form inputs (text, email, password, select, radio, checkbox)
- Rating systems (star ratings)
- Card-based information display

This wireframe document provides a comprehensive view of all 11 HTML pages' layout structures and their relationships within the CHARUSAT University web application.
